# Руководство по падежам / Russian Cases Guide

## Обзор

Бот теперь поддерживает все 6 падежей русского языка для грамматически правильных сообщений!

## Падежи русского языка

1. **Именительный** (Nominative) - Кто? Что? → Япония
2. **Родительный** (Genitive) - Кого? Чего? → Японии
3. **Дательный** (Dative) - Кому? Чему? → Японии
4. **Винительный** (Accusative) - Кого? Что? → Японию
5. **Творительный** (Instrumental) - Кем? Чем? → Японией
6. **Предложный** (Prepositional) - О ком? О чём? → Японии

## Команды

### `/add <слово>`
Добавляет новое слово для отслеживания.

**Пример:**
```
/add Токио
```

По умолчанию все падежи будут равны именительному падежу. Используйте `/setcase` для настройки.

### `/setcase <слово> <Род> <Дат> <Вин> <Твор> <Пред>`
Устанавливает падежные формы для слова.

**Формат:**
```
/setcase Слово Родительный Дательный Винительный Творительный Предложный
```

**Пример для слова "Япония":**
```
/setcase Япония Японии Японии Японию Японией Японии
```

**Можно пропустить падежи с помощью "-":**
```
/setcase Япония Японии - Японию - -
```
Это обновит только родительный и винительный падежи.

### `/list`
Показывает все отслеживаемые слова с их падежными формами.

**Пример вывода:**
```
Отслеживаемые слова:
• Япония (Р:Японии, В:Японию, Т:Японией, П:Японии)
• Токио

И=именительный, Р=родительный, Д=дательный, В=винительный, Т=творительный, П=предложный
```

## Где используются падежи в сообщениях

### Винительный падеж (после "про"):
- "Красавчики опять заговорили про **Японию**"
- "Эх, сколько раз уже упомянули **Японию**"

### Предложный падеж (после "о"):
- "Е-маё, все мечтаем о **Японии**"

### Именительный падеж (подлежащее):
- "Так, ещё одно упомянутие **Япония**"
- "**Япония** в этом чате упомянули уже 58 раз"

## Примеры для разных слов

### Япония
```
/setcase Япония Японии Японии Японию Японией Японии
```

### Токио (не изменяется)
```
/add Токио
```
Падежи настраивать не нужно - слово не склоняется.

### Киото
```
/add Киото
```
Также не склоняется.

### Осака
```
/setcase Осака Осаки Осаке Осаку Осакой Осаке
```

## Технические детали

- Все падежные формы хранятся в файле `tracked_keywords.json`
- Если падеж не указан, используется именительный падеж (nominative)
- Старые ключевые слова автоматически мигрируются в новый формат
- Бот автоматически выбирает правильный падеж для каждой фразы
